Rate Ninja API

The browser uses same-origin endpoints. Airtable is intentionally not exposed to the browser.

EndpointAccessPurpose
POST /api/auth/loginPublicCreates an HTTP-only session from a username and password.
POST /api/auth/logoutSigned inEnds the current session.
GET /api/sessionSigned inReturns the signed-in user's display name and administrator status.
GET /api/ratesSigned inReturns rates available to the user's RateView with the appropriate margin applied.
GET /api/sailingsSigned inReturns sailings for carrier, origin port, and effective-date query parameters.
GET /api/admin/companiesAdministratorReturns editable companies in the administrator's RateView.
PATCH /api/admin/companies/:recordIdAdministratorUpdates a company's margin percent and fixed margin.
GET /api/v1/ratesDemo API keyReturns base rates with optional carrier, originPort, destinationPort, page, and pageSize filters.
GET /api/v1/sailingsDemo API keyReturns sailings for carrier, originPort, and after query parameters.
GET /api/v1/predictive-pricingDemo API keyReturns stub predictive prices for carrier, originPort, and an after date more than 90 days out.

The v1 demo API requires X-API-Key, returns base rates only, and is limited to 60 requests per minute per running server instance. See API-README.md for examples. Do not put an Airtable personal access token in a browser, this page, or source control.
